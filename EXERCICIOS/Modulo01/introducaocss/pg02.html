CSS inline

Nesse caso seu arquivo HTML se tornará muito grande e cansativo para ler. Caso seu cliente deseje alterar algum estilo do produto, você terá que mudar cada elemento, tornando seu trabalho mais demorado e cansativo.****

CSS interno

Nesse caso o estilo CSS que será adicionado estará dentro do html porém na área head, dentro da tag <style>. Sempre que o cliente quiser alterar o estilo do produto, você terá que açterar o estilo dentro de cada página.

CSS externo

Essa é a maneira mais correta de adicionar estilos a uma página. Para alterar os estilos do produto bastará acessar o arquivo em CSS externo e relalizar as alterações desejadas pelo cliente.
